## Style

`*` 与 `~` 都表示特殊字体

- <i>斜体</i> (`*斜体*`)
- <b>粗体</b> (`**粗体**`)
- <mark>百科</mark> (`***高亮***`)
- <u>下划线</u> (`~下划线~`)
- <del>删除线</del>(`~~删除线~~`)
- *斜体* (`*斜体*`)
